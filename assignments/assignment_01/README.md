# Assignment 01 - Reproducible Project Structure

This README is a complete command log documenting how the
Assignment 01 directory was constructed using only the  terminal
All commands were executed manually,
and are listed in the exact order they were run. Running these
commands line-by-line will recreate the same project structure.

# Navigate to assignments directory
cd assignments

# Navigate into Assignment 01 directory
cd assignment_01

# Create data directory
mkdir data

# Create raw data directory
mkdir data/raw

# Create cleaned data directory
mkdir data/cleaned

# Create raw data placeholder script
touch data/raw/example_raw_data.sh

# Create cleaned data placeholder script
touch data/cleaned/example_cleaned_data.sh

# Create scripts directory
mkdir scripts

# Create output directory
mkdir output

# Create documentation directory
mkdir docs

# Add placeholder documentation files
touch docs/example_docs.md
touch docs/example_docs.sh

# Create additional markdown example files
touch example_01.md
touch example_02.md
touch example_03.md

# Create README file
touch README.md

# Create essay file
touch assignment_1_essay.md
